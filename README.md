🎹 Piano

Piano é uma aplicação em Python que reproduz automaticamente músicas no site Virtual Piano, simulando as teclas do teclado.

O usuário apenas cola uma partitura no formato do Virtual Piano, escolhe o intervalo entre as notas e a aplicação executa a música automaticamente.

Funcionalidades
🎼 Suporte à notação do Virtual Piano
🎹 Reprodução de notas individuais e acordes
⚡ Intervalo entre notas configurável
🖥️ Interface simples desenvolvida com PySide6
🌐 Abre automaticamente o site Virtual Piano
⌨️ Simula a digitação das teclas utilizando PyAutoGUI
Como funciona
Cole uma partitura do Virtual Piano na aplicação.
Escolha o intervalo entre as notas.
Clique em Iniciar.
A aplicação abre o Virtual Piano.
Após uma breve contagem regressiva, a música começa a ser executada automaticamente.
Exemplo

Entrada:

[6eu] p s u p s
u p s u p s
[5wu] p s u p s

A aplicação interpreta a notação e reproduz automaticamente a música no site Virtual Piano.

Tecnologias utilizadas
Python
PySide6
PyAutoGUI
Planos futuros

O objetivo de longo prazo deste projeto é evoluir de um simples reprodutor automático para um interpretador musical assistido por Inteligência Artificial.

As próximas versões poderão incluir:

🎵 Geração automática do tempo entre as notas por IA
🎼 Reprodução dinâmica do ritmo original da música
📁 Suporte a arquivos MIDI
📖 Reconhecimento de partituras
🎤 Conversão de áudio para partitura
⏱️ Detecção automática de andamento (BPM)
🧠 Reprodução inteligente, buscando aproximar a execução do ritmo e da interpretação originais
